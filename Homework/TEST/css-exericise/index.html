הסבר על CSS

CSS Location

מיקום של קבצי ה-CSS יכול להיות בשני אופנים עיקריים:

  • Inline CSS: כאשר ההגדרות נכתבות בתוך האלמנט עצמו בתוך קוד ה-HTML (לדוגמה: <h1 style="color: blue;">).
  • Internal CSS: כאשר ההגדרות נכתבות בתוך תגית <style> בתוך קובץ ה-HTML.
  • External CSS: כאשר ההגדרות נמצאות בקובץ נפרד עם סיומת .css ומקושרות לקובץ ה-HTML באמצעות <link>.

דוגמה ל-External CSS:

<link rel="stylesheet" href="styles.css">

CSS Selector

סלקטור ב-CSS הוא כלי שמאפשר לנו לבחור אלמנטים מסוימים ב-HTML וליישם עליהם עיצוב. קיימים סוגים רבים של סלקטורים, כמו סלקטור לפי תגית, מחלקה, מזהה, או סלקטורים מורכבים יותר.

דוגמה לסלקטור לפי מחלקה:

.example-class { color: red; }

CSS Selector Exercise

תרגול של סלקטורים ב-CSS כולל כתיבת סלקטורים שונים ולמידה כיצד לבחור אלמנטים בצורה מדויקת, כגון לפי תגית, מחלקה, מזהה, או שילובים שונים של סלקטורים.

דוגמה לתרגול:

<div class="container">
  <p id="paragraph">This is a paragraph.</p>
  <span class="highlight">Highlight this text.</span>
</div>

סלקטורים לדוגמה עבור התרגול:

p { color: blue; } /* בחירת כל הפסקאות */
#paragraph { font-weight: bold; } /* בחירת אלמנט לפי מזהה */
.highlight { background-color: yellow; } /* בחירת אלמנט לפי מחלקה */
